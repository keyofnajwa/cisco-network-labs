# 🌐 Cisco Networking Academy - Module Documentation

Repositori ini berisi kumpulan dokumentasi, catatan materi, topologi, dan file simulasi (*lab*) dari seluruh modul jaringan Cisco yang telah **selesai dikerjakan**. Proyek ini berfungsi sebagai arsip pribadi sekaligus portofolio pemahaman jaringan komputer saya.

## 📌 Status Proyek: Selesai (Completed)

Semua tugas, praktikum, dan materi dalam modul ini telah diselesaikan dengan baik menggunakan alat simulasi utama.

- **Alat Utama:** Cisco Packet Tracer 
- **Kompetensi yang Dicapai:** Konfigurasi perangkat dasar, manajemen IP Address (Subnetting), Routing (Static & Dynamic), 
## 📂 Struktur Dokumentasi

Berikut adalah gambaran umum materi yang telah didokumentasikan di dalam repositori ini:

* **Dasar Jaringan:** Konfigurasi awal router/switch, manajemen iOS, dan skema pengalamatan IPv4/IPv6.
* **Routing Protocols:** Konfigurasi routing statis serta protokol routing dinamis seperti OSPF.
* **Network Services & Security:** Pengamanan jalur akses (SSH), dan konfigurasi Network Address Translation (NAT).
